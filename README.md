# 🔧 Command Block Blue Furnace - MCPE 26.33+

## ✨ Características

✅ **Craft com 9 pedras** - Recipe simples na mesa de trabalho  
✅ **Textura de fornalha azul** - Design exclusivo  
✅ **Menu de comandos interativo** - Interface amigável com formulários  
✅ **SEM desativar conquistas** - Funciona sem ativar trapaças!  
✅ **Comandos Bedrock completos** - Todos os comandos do Minecraft Bedrock  
✅ **Suportado em MCPE 26.33+** - Compatibilidade garantida  

## 🎮 Como Instalar

### Opção 1: Download como .mcaddon
1. Clique em **"Code"** no repositório
2. Clique em **"Download ZIP"**
3. Extraia o arquivo
4. Comprima novamente sem a pasta raiz em um ZIP
5. Renomeie de `.zip` para `.mcaddon`
6. Toque no arquivo no Minecraft PE para instalar

### Opção 2: Instalar direto do repositório
1. Abra o Minecraft PE
2. Vá em Configurações → Comportamento
3. Adicione os arquivos deste repositório

## 📋 Como Usar

1. **Craftar o item:**
   - Use 9 blocos de **pedra** em um padrão 3x3 na mesa de trabalho
   - Você receberá o **Bloco de Comando - Fornalha Azul**

2. **Usar o item:**
   - Segure o item na mão
   - Clique com o botão direito (ou toque)
   - Abrirá o menu de comandos

3. **Selecionar um comando:**
   - Uma interface irá aparecer
   - Escolha o comando desejado
   - Preencha os dados necessários
   - Execute!
